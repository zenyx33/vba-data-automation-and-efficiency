# VBA Process Automation: 77% Efficiency Increase in Banking Operations

Este projeto apresenta uma solução de automação desenvolvida para otimizar o processamento de demandas judiciais em uma grande operação bancária jurídica.

## 🚀 Impacto Real
* **Redução de Tempo:** De 4 minutos para 55 segundos por tarefa.
* **Ganho de Produtividade:** Economia de **17 horas úteis diárias** para a equipe de operações.
* **Compliance:** Garantia de 100% de conformidade com prazos legais e eliminação de erros manuais.

## 🛠️ Tecnologias e Técnicas
* **VBA (Visual Basic for Applications):** Automação core.
* **Cross-App Integration:** Integração dinâmica entre Excel, Outlook (envio/leitura) e Word (conversão de formatos).
* **Data Logging:** Sistema de log automático para auditoria de envios.
* **Error Handling:** Tratamento de exceções para garantir a continuidade do processamento em lote.

## 📂 Estrutura do Repositório
* `EmailAutomation.vba`: Motor de envio de e-mails e gestão de templates.
* `PDFConverter.vba`: Script inteligente para conversão de e-mails em PDF via Word Interop.
* `FolderManager.vba`: Organização automática de diretórios baseada em dados de planilhas.
